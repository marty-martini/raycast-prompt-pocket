# Raycast Store Screenshots Guide

このフォルダには Raycast Store 公開用のスクリーンショットを配置します。

## 📸 必要なスクリーンショット

### 推奨枚数: 3-5枚

Raycast Store では **1-8枚** のスクリーンショットを提出できますが、**3-5枚が推奨**です。

### スクリーンショット要件

- **サイズ**: 幅 1600px 以上推奨
- **フォーマット**: PNG または JPG
- **アスペクト比**: 16:9 または 4:3
- **解像度**: Retina ディスプレイに対応した高解像度
- **モード**: ライト/ダークモード両方を含めることを推奨

---

## 🎯 推奨スクリーンショット内容

Extension の主要機能を順序立てて紹介する構成が効果的です。

### 1. **プロンプト一覧画面**（必須）
- **ファイル名**: `prompt-pocket-1.png`
- **内容**: 
  - 複数のプロンプトが表示された一覧画面
  - タグが表示されている状態
  - 最終使用日時が表示されている
  - 検索バーが見える状態
- **目的**: Extension の基本的なUIと機能を紹介

### 2. **検索機能のデモ**（推奨）
- **ファイル名**: `prompt-pocket-2.png`
- **内容**:
  - 検索バーに検索キーワードを入力した状態
  - 検索結果が絞り込まれている様子
  - "Search prompts by title, body, or tags..." というプレースホルダーが見える
- **目的**: 検索機能の便利さをアピール

### 3. **プロンプト作成/編集画面**（推奨）
- **ファイル名**: `prompt-pocket-3.png`
- **内容**:
  - プロンプトの作成または編集フォーム
  - Title、Body、Tags の各フィールドが入力されている
  - プレースホルダー `{clipboard}` や `{cursor}` を使用した例
- **目的**: プロンプト作成の簡単さを示す

### 4. **プレースホルダー機能のデモ**（推奨）
- **ファイル名**: `prompt-pocket-4.png`
- **内容**:
  - プレースホルダー（`{clipboard}`, `{cursor}`）を含むプロンプト
  - アクションパネルに "Paste Filled Prompt to Active App" などのアクションが表示されている
  - 詳細画面でプレースホルダーの説明が見える
- **目的**: Extension のユニークな機能（プレースホルダー）をアピール

### 5. **詳細表示画面**（オプション）
- **ファイル名**: `prompt-pocket-5.png`
- **内容**:
  - プロンプトの詳細表示画面
  - メタデータ（作成日時、更新日時、最終使用日時）が表示されている
  - プレースホルダーの説明が表示されている
- **目的**: 詳細な管理機能を示す

## 🎨 スクリーンショット作成のコツ

### 1. **実用的なサンプルデータを使用**
- テスト用のダミーデータではなく、実際に使いそうな内容を作成
- 例: "Bug Report Template", "Email Reply Template", "Code Review Checklist" など

### 2. **ライトモードとダークモード**
- 可能であれば両方のモードのスクリーンショットを含める
- Raycast のテーマは Command + Option + T で切り替え可能

### 3. **テキストの可読性**
- スクリーンショットのテキストが読みやすいサイズであることを確認
- Raycast の拡大表示を使用して、適切なサイズで撮影

### 4. **実際の使用シーンを想像**
- ユーザーが「この Extension を使うとどうなるか」をイメージできる内容
- ビフォー・アフターを示すと効果的

## 📝 スクリーンショットの撮影方法

### macOS の標準機能を使用

1. **Command + Shift + 4** を押す
2. **スペースキー** を押してウィンドウキャプチャモードに切り替え
3. Raycast のウィンドウをクリックして撮影

### Raycast の推奨設定

撮影前に以下の設定を確認：
- Raycast のウィンドウサイズを適切に調整
- フォントサイズが読みやすいことを確認
- 不要な通知やポップアップを閉じる

## ✅ チェックリスト

撮影後、以下を確認してください：

- [ ] すべてのスクリーンショットが 1600px 以上の幅である
- [ ] PNG または JPG フォーマットである
- [ ] テキストが鮮明に読める
- [ ] Extension の主要機能が網羅されている
- [ ] ファイル名が `prompt-pocket-1.png` のような規則に従っている
- [ ] 個人情報や機密情報が含まれていない

## 🚀 スクリーンショット配置後

スクリーンショットを配置したら、以下のコマンドで公開プロセスを開始できます：

```bash
npm run publish
```

このコマンドは自動的に：
1. スクリーンショットを検出
2. package.json の情報を読み込み
3. Raycast の Extensions リポジトリにプルリクエストを作成
